## 📋 Quick Start Guide

### Step 1 — Run as Administrator
> [!IMPORTANT]
> **Administrator privileges are mandatory!** Right-click `VetBot.exe` and select **Run as Administrator** to allow hardware-level input simulation to interact with the active game window.

### Step 2 — Activate Your License
1. On first launch, the **Activation Wizard** will open.
2. Select your desired plan and fill in your details to generate a request file (`.vreq`).
3. Send the `.vreq` file to the developer via Discord or Telegram.
4. Once processed, load your unique license key (`.vlic`) to unlock VetBot.

### Step 3 — Select & Lock Game Window
1. Open *Tantra Online* and log into your character.
2. Inside VetBot, click **Select Target** and hover over your game window.
3. VetBot will instantly bind its vision engine and input stream to that client.

### Step 4 — Configure & Automate
* **Scanner Region:** Set the screen detection zone where monsters appear.
* **Buff Configuration:** Load pre-cropped icons (`assets/buff_icons/`) for automated skill upkeep.
* **Combat Rotation:** Set up custom key sequences, attack loops, and skill spam throttles.
* **Smart Safeguards:** Configure automated HP/MP potion thresholds and disconnect triggers.
* Click **START** and let VetBot run smoothly 24/7!
